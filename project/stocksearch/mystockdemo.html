Stock Search Application Demo


Date Apr 2016 to May 2016
Skills HTML5, CSS3, Javascript, PHP, JSON, jQuery, AJAX, Bootstrap, Facebook API, Amazon Web Services (AWS), Google App Engine (GAE), Android Studio, Java

This Stock Search app is a smart tool for users to search and track stock information. Users can add items into their favorite list for easy tracking. In addition, there is an automatic refresh toggle switch which will refresh the price every 5 seconds. In this app, users can easily find the related charts and news of specific stock. It also allows users pubilsh stock information on Facebook. Currently, I've finished both a website version and a mobile version. Please see demos below.
